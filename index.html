Success Stories

On my lead, we switched from C# applications on Windows to embedded Linux with Python, Docker, React and Tornado web server. I did work a long time with the old technology stack and found it not optimal for our use. When we started to develop a new product, which needed HW controlling, I saw my chance to do the switch. Luckily I got my proposal through on management (that was the hard part!). It was supposed to be a controlling system just for that product, but like all the best platforms, the thing we created proved to be so usable that it grew into a HW controlling platform. Now it is used in multiple products and has made new product designing much faster!

Key things I changed were:

  • Use open source components where possible to reduce the codebase we maintain.
  • Use modern tools/platforms like React and Docker.
  • Make RestAPI with HATEOAS to make client software extremely thin.
  • Switch to Linux, which should be obvious when we are talking about the connected embedded devices.
  • Use CI pipeline to automate testing and deployment.
